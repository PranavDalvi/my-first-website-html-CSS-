- March 05, 2015


Important Note: VB is listed in Dying list of programing language list so Please don't Learn if you want job in VB.


Visual Basic is a third-generation event-driven programming language from Microsoft for its Component Object Model (COM) programming model first released in 1991 and declared legacy during 2008. Microsoft intended Visual Basic to be relatively easy to learn and use. Visual Basic was derived from BASIC and enables the rapid application development (RAD) of graphical user interface (GUI) applications, access to databases using Data Access Objects, Remote Data Objects, or ActiveX Data Objects, and creation of ActiveX controls and objects. (from Wikipedia)

System Specs.

From Microsoft®

  • 1.8 GHz or faster processor. Dual-core or better recommended
  • 2 GB of RAM; 4 GB of RAM recommended (2.5 GB minimum if running on a virtual machine)
  • Hard disk space: up to 130 GB of available space, depending on features installed; typical installations require 20-50 GB of free space.
  • Hard disk speed: to improve performance, install Windows and Visual Studio on a solid state drive (SSD).
  • The video card that supports a minimum display resolution of 720p (1280 by 720) ; Visual Studio will work best at a resolution of WXGA (1366 by 768) or higher.

I am going to show you how to install V.B and run first program (Hello World) in visual basic (V.B)



1. INSTALLATION OF VISUAL STUDIO:


1. Download Visual Studio from official Microsoft® website.

2. Select versions as per your requirements.


3. After installation of Visual Studio Installer click on install after ticking .NET desktop development.


2. Creating First V.B Project


1. Click on File located in title-bar, a drop-down menu will appear, now select "New, Project.


2. Select the Visual Basic from the installed located left side.
3. Now select Windows Form APP (.NET Framework).
4. Type the name of the file you like and click on OK button.


3. GETTING BASICS


Form: Form is the place where we create our User Interface (UI) for our program.
Toolbox: toolbox is the place of the all tools which is necessary to create program's UI. e.g. Text, Label, Button etc.
Properties: Properties is used to change NAME, TEXT, CAPTION which are important to differentiate between the Buttons and NAME is used in coding. So better keep that in mind.


4. GET SET GO!!


1. To add the iterm on the form just drag the items or double click on the items.


2. change the text of the button. e.g. Click me for message for Button1 and Clear for Button2


3. Repeat 1st and 2nd steps for Button2.


4. Now change Name of buttons (don't get confuse with Name and Text- Name is real name which is used in coding & text is the UNDERCOVER name which is not used in coding & it is used only for representation of buttons & label)
5.Now repeat 4th step for Button2.


6. Now change Name of Text-box and keep Text blank in properties or there will be no output (specially for V.B 6.0).


5. Lets DO CODING..CODING..CODING......


1. Now double click on the both buttons one by one.


2. Now coding window will pop up already with basic code & you can see the name of the buttons are appeared in the code.


3. Now type the following code in the blank space in between the code for cmdmsg: Text1.Text = "HELLO WORLD" & for cmdclear: Text1.Text = " " & for cmdexit: End (optional)
4. tap on tab button to auto fill code highlighted in the drop down menu.


5. click on start button if your coding is done.

Source Code

Public Class Form1
    Private Sub cmdmsg_click()
        Text1.Text = "HELLO WORLD"
    End Sub

    Private Sub cmdclear_click()
        Text1.Text = " "
    End Sub

    Private Sub cmdexit_click()
        End
    End Sub
End Class
